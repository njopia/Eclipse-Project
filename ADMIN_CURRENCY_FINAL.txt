¿QUÉ SE HA CREADO?
═════════════════════════════════════════════════════════════════════════════

Un sistema completo de menú para administradores que permite dar currency
(puntos) a jugadores elegidos del servidor de forma interactiva.

CARACTERÍSTICAS PRINCIPALES:
  ✓ Menú interactivo de selección de jugadores
  ✓ Montos preestablecidos (25, 50, 100, 250, 500, 1000)
  ✓ Cantidad personalizada (admin puede ingresar cualquier monto)
  ✓ Usa SetPlayerCurrency() para establecer el balance
  ✓ Solo accesible por administradores (ADMFLAG_GENERIC)
  ✓ ConVar para habilitar/deshabilitar el sistema
  ✓ Logs detallados de todas las transacciones
  ✓ Validaciones de seguridad en todos los pasos

CÓMO SE USA
═════════════════════════════════════════════════════════════════════════════

PARA EL ADMINISTRADOR:

1. En el chat del servidor o consola:
   Escribir: !givecurrency

2. Se abrirá un menú mostrando todos los jugadores sobrevivientes

3. Seleccionar el jugador a quien dar currency

4. Elegir cantidad:
   - Cantidad fija (25, 50, 100, 250, 500, 1000)
   - O "Cantidad personalizada"

5. Si eligió personalizada:
   Escribir en chat: !givecustom <número>
   Ejemplo: !givecustom 750

6. Sistema confirma y notifica al jugador

═════════════════════════════════════════════════════════════════════════════
EJEMPLO PRÁCTICO
═════════════════════════════════════════════════════════════════════════════

ESCENARIO: Admin quiere dar 500 puntos a "Player1" como recompensa

1. Admin en el servidor escribe:
   !givecurrency

2. Se abre menú con jugadores conectados:
   [Player1]      ← Admin selecciona
   [Player2]
   [Player3]
   [Player4]

3. Se abre segundo menú:
   ¿Cuántos puntos dar a Player1?

   [25 puntos]
   [50 puntos]
   [100 puntos]
   [250 puntos]
   [500 puntos]   ← Admin selecciona
   [1000 puntos]
   [Cantidad personalizada]

4. Sistema ejecuta:
   • Obtiene currency actual de Player1: 250
   • Calcula nuevo total: 250 + 500 = 750
   • Establece balance: SetPlayerCurrency(Player1, 750)

5. Notificaciones:
   Admin ve: "[Admin] ✓ Diste 500 puntos a Player1 (Total: 750)"
   Player1 ve: "[Admin] El admin AdminName te dio 500 puntos (Total: 750)"
   Log: "ADMIN_CURRENCY: AdminName gave 500 points to Player1 (new total: 750)"
